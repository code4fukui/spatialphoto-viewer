# spatialphoto-viewer
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Spatial photo viewer for Quest/Vision Pro with Three.js.

## Demo
https://code4fukui.github.io/spatialphoto-viewer/

## Features
- Displays a collection of spatial photos in a 3D scene
- Supports both VR and AR viewing modes
- Automatically adjusts positioning and orientation of the photos
- Integrates with the WebXR API for immersive experiences

## Requirements
- A WebXR-enabled browser (e.g., Chrome, Safari)
- A VR headset (e.g., Quest, Vision Pro) or AR-compatible device

## Usage
1. Open the [demo page](https://code4fukui.github.io/spatialphoto-viewer/) in a WebXR-enabled browser.
2. Click the "START AR" or "START VR" button to enter the respective viewing mode.
3. Interact with the spatial photos by looking around and moving your head.
