# spatialphoto-viewer
English README is available here: [README.md](README.md)

Three.jsを使ったQuest/Vision Pro向けの空間写真ビューアー。

## デモ
https://code4fukui.github.io/spatialphoto-viewer/

## 機能
- 3Dシーンに空間写真の収集を表示します
- VRおよびARの表示モードをサポートしています
- 写真の位置と向きを自動的に調整します
- WebXR APIを使用してイマーシブな体験を提供します

## 要件
- WebXRに対応したブラウザ (例: Chrome、Safari)
- VRヘッドセット (例: Quest、Vision Pro) またはAR対応デバイス

## 使い方
1. WebXRに対応したブラウザで[デモページ](https://code4fukui.github.io/spatialphoto-viewer/)を開きます。
2. "START AR"または"START VR"ボタンをクリックして、それぞれのビューモードに入ります。
3. 視線を動かしたり頭を動かしたりして、空間写真と対話します。
